# Andor PyMeasure Legacy

This repository contains legacy PyMeasure and Qt-based implementations for Andor spectrometer control.

## Packages

- **andor_pymeasure** - PyMeasure-based procedures for spectrum acquisition
- **andor_qt** - Qt/PySide6 GUI with PyMeasure integration

## Note

This code has been archived from the main `andor-chimera` repository. The active development continues in the Tkinter-based `andor-chimera` package.

## Installation

```bash
uv pip install -e .
```

## Usage

```bash
# PyMeasure CLI
andor-pymeasure

# Qt GUI
andor-qt
```
